JavaScript组成

JavaScript是一种专为与网页交互而设计的脚本语言，由下列三个不同的部分组成：

  • ECMAScript （核心）
  • DOM （文档对象模型）
  • BOM （浏览器对象模型）

  • 不能把关键字、保留字、true、false和null用作标识符

关键字和保留字

关键字：用于表示控制语句的开始或结束，或用于执行特定操作等，不能用作标识符

保留字：将来有可能会被用作关键字

变量

ECMAScript的变量是松散类型的，可以用来保存任何类型的数据
定义变量要使用var操作符

注意：使用var操作符定义的变量将成为定义该变量的作用域中的局部变量

数据类型*

ECMAScript有5种简单数据类型（基本数据类型）：

  • Undefined
  • Null
  • Boolean
  • Number
  • String

1种复杂数据类型：

  • Object

typeof操作符

用于检测给定变量的数据类型，对一个值使用typeof操作符可能返回下列某个字符串：

  • “undefined” ———— 该值未定义
  • “boolean” ———— 该值是布尔值
  • “string” ———— 该值是字符串
  • “number” ———— 该值是数值
  • “object” ———— 该值是对象或null
  • “function” ———— 该值是函数

Undefined类型

该类型只有一个值即undefined，在使用var声明变量但未对其初始化时，这个变量的值就是undefined

Null类型

该类型只有一个值即null，从逻辑角度，null值表示一个空对象指针，所以使用typeof操作符检测null值会返回“object”
实际上，undefined值是派生自null值的，因此ECMA-262规定对它们相等性测试要返回true

Boolean类型

该类型有两个字面值：

  • true
  • false

注意：true和false区分大小写，True和False或其他混合大小写的形式都不是Boolean值，只是标识符

ECMAScript中所有类型的值都有与这两个Boolean值等价的值，可以调用转型函数Boolean()

数据类型 转换为true的值 转换为false的值
Boolean true false
String 任何非空字符串 “”( 空字符串 )
Number 任何非零数字值(包括无穷大) 0和NaN
Object 任何对象 null
Undefined n/a(不适用) undefined

Number类型

多种数值字面量：

  • 十进制整数

  • 八进制（第一位必须是0，然后是八进制数字序列(0~7)，字面值中的数值超出了范围，前导零将被忽略）

  • 十六进制（前两位必须是0x，后跟任何十六进制数字(0~9及A~F)，其中A~F可大写或小写）

注意：进行算术计算时，所有八进制和十六进制表示的数值都将被转换成十进制数值

浮点数值

数值中包含一个小数点，并且小数点后必须至少有一位数字

对于极大或极小的数值可以用e表示法（即科学记数法）表示的浮点数值表示：

注意：浮点数值的最高精度是17位小数，但在进行算术计算时其精确度远远不如整数，例如，0.1加0.2结果不是0.3，而是0.30000000000000004，这个小小的舍入误差会导致无法测试特定的浮点数值

数值范围

由于内存的限制，ECMAScript并不能保存世界上所有的数值

  • 最小数值保存在Number.MIN_VALUE中 ———— 大多数浏览器中这个值是5e-324
  • 最大数值保存在Number.MAX_VALUE中 ———— 大多数浏览器中这个值是1.7976931348623157e+308

如果某次计算结果超出JavaScript数值范围，这个数值将被转换成Infinity（正无穷）或-Infinity（负无穷），该值将无法继续参与下一次的计算，可以使用isFinite()函数确定一个数值是否无穷，参数位于最小与最大数之间会返回true：

字符串是不可变的，一旦创建，值就不能改变

转换为字符串

把一个值转换为字符串：

  • String() ———— 将任何值转换成字符串，有toString()方法的调用toString()方法，值是null返回“null”，值是undefined返回“undefined”

位操作符

按内存中表示数值的位来操作数值

ECMAScript中的所有数值都以IEEE-754 64位格式存储，位操作符将64位的值转换成32位的整数再将结果转换回64位
对于有符号的整数，前31位表示整数的值，第32位是符号位，0表示正数，1表示负数

  • 正数：纯二进制格式存储，空位填0
  • 负数：二进制补码格式存储
    • 求这个数值绝对值的二进制码
    • 求反码，即0替换1，1替换0
    • 反码加1

注意：在处理有符号整数时，是不能访问位31的

副效应：对特殊的NaN和Infinity值应用位操作时，都会被当成0来处理

乘性操作符

乘法

（*）表示乘法操作符，用于计算两个数值的乘积

在处理特殊值的情况下，遵循如下规则：

  • 操作数都是数值正常计算，一个操作数有符号结果为负数，乘积超过数值范围返回Infinity或-Infinity
  • 有一个操作数是NaN ———— 返回NaN
  • Infinity与0相乘 ———— 返回NaN
  • Infinity与非0数值相乘 ———— 返回Infinity或-Infinity
  • Infinity与Infinity相乘 ———— 返回Infinity
  • 有一个操作数不是数值 ———— 调用Number()转换为数值再应用上面的规则

除法

（/）表示除法操作符，计算第二个操作数除第一个操作数

在处理特殊值的情况下，遵循如下规则：

  • 操作数都是数值正常计算，一个操作数有符号结果为负数，乘积超过数值范围返回Infinity或-Infinity
  • 有一个操作数是NaN ———— 返回NaN
  • Infinity被Infinity除 ———— 返回NaN
  • 0被0除 ———— 返回NaN
  • 非0的有限数被0除 ———— 返回Infinity或-Infinity
  • Infinity被任何非0数值除 ———— 返回Infinity或-Infinity
  • 有一个操作数不是数值 ———— 调用Number()转换为数值再应用上面的规则

求模

（%）表示求模（余数）操作符

在处理特殊值的情况下，遵循如下规则：

  • 操作数都是数值 ———— 常规除法计算，返回除得的余数
  • 被除数是无穷大值，除数是有限大的数值 ———— 返回NaN
  • 被除数是有限大的数值，除数是0 ———— 返回NaN
  • Infinity被Infinity除 ———— 返回NaN
  • 被除数是有限大的数值，除数是无穷大值 ———— 返回被除数
  • 被除数是0 ———— 返回0
  • 有一个操作数不是数值 ———— 调用Number()转换为数值再应用上面的规则

关系操作符

小于（<）、大于（>）、小于等于（<=）、大于等于（>=）用于对两个值进行比较，结果都返回布尔值

比较操作数遵循如下规则：

  • 两个操作数都是数值 ———— 执行数值比较
  • 两个操作数都是字符串 ———— 比较两个字符串对应的字符编码值
  • 一个操作数是数值 ———— 将另一个操作数转换为一个数值，执行数值比较
  • 一个操作数是对象 ———— 调用valueOf()用得到的结果按照上面的规则执行比较；没有valueOf()方法则调用toString()将得到的结果按照上面的规则执行比较

相等操作符

相等和不相等 —— 先转换再比较

（==）表示相等操作符，两个操作数相等返回true；（!=）表示不相等操作符，两个操作数不相等返回true

转换不同的数据类型时，遵循如下规则：

  • 一个操作数是布尔值 ———— 比较相等性之前先将其转换为数值，false转换为0，true转换为1
  • 一个操作数是字符串，另一个是数值 ———— 比较相等性之前先将其转换为数值
  • 一个操作数是对象，另一个不是 ———— 调用valueOf()用得到的基本类型值按照上面的规则执行比较

这两个操作符在进行比较时要遵循如下规则：

  • null和undefined是相等的
  • 要比较相等性之前，不能将null和undefined转换成其他任何值
  • 一个操作数是NaN，相等操作符返回false，不相等操作符返回true（两个操作数都是NaN相等操作符也返回false，NaN不等于NaN）
  • 两个操作数都是对象，比较它们是不是同一个对象，都指向同一个对象则相等操作符返回true，否则返回false

特殊情况及比较结果：

NaN != NaN true
false == 0 true

true == 2 false
undefined == 0 false
null == 0 false
“5” == 5 true
